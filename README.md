# MediaPipe Real-Time Detection

## Overview

This project demonstrates real-time computer vision using **MediaPipe**, an open-source framework developed by **Google**. The system captures video from a webcam and processes it to detect human features such as hands, face landmarks, or body pose.

MediaPipe provides highly optimized machine learning pipelines that work efficiently in real time, making it suitable for applications such as gesture recognition, virtual interaction, and augmented reality.

## Features

* Real-time webcam processing
* Hand landmark detection
* Efficient ML pipeline using MediaPipe
* Simple and lightweight implementation
* Easy to extend for pose detection or face tracking

## Technologies Used

* Python
* OpenCV
* MediaPipe

## Installation

1. Clone the repository

```
git clone https://github.com/your-username/mediapipe-detection.git
```

2. Navigate to the project directory

```
cd mediapipe-detection
```

3. Install required dependencies

```
pip install opencv-python mediapipe
```

## Usage

Run the Python script:

```
python main.py
```

The webcam will start and detect hand landmarks in real time.

Press **Q** to exit the application.
